    - Client Data : @ModelAttribute / @RequestParam 적절이 사용

* session 내용 다시 복습
```
setAttribute(String name, Object value)
```
- setAttribute는 name, value 쌍으로 객체 Object를 저장하는 메서드 세션이 유지되는 동안 저장
```
session.setAttribute(이름, 값)
```
- 이렇게 사용할 수 있다.

* session값 조회
```
getAttribute(String name)
```
